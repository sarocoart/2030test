# 2030 Platform Charter

Version 1.0

**Build Once. Operate Forever.**

---

# Mission

We are not building a single personality test.

We are building **2030 Platform OS**, a reusable AI-powered platform that will support multiple products, including:

* Personality Tests
* LUMI
* Cognitive Training
* Dementia Prevention
* AI Glass
* Community
* AI Recommendations
* Future AI Services

Every new feature should strengthen the platform instead of solving only one project.

---

# Core Philosophy

## 1. CMS First

Never hardcode business content.

Administrators should manage everything from Google Sheets.

---

## 2. Feature Toggle

Every feature must support:

TRUE / FALSE

Administrators should be able to show or hide features without editing code.

---

## 3. Google Sheets Is The CMS

Google Sheets

↓

Apps Script

↓

JSON API

↓

Website

Google Sheets is the administrator panel.

---

## 4. Everything Is Data

Treat everything as editable data.

Including:

* Text
* Images
* Buttons
* Menus
* Advertisements
* Recommendations
* Notices
* Events
* Colors
* Settings

---

## 5. Mobile First

Every feature must work on:

* Desktop
* Tablet
* Mobile

---

## 6. User Experience First

Advertisements should feel like recommendations.

Never interrupt users.

---

## 7. Reusable Modules

Every feature should be reusable across future projects.

Avoid project-specific code whenever possible.

---

## 8. Future Ready

The architecture should allow future migration from:

Google Sheets

↓

Supabase

↓

PostgreSQL

↓

BigQuery

↓

Redis

without major frontend changes.

---

# Platform Principles

Before implementing a feature, always ask:

1. Does it provide value to users?

2. Can administrators manage it through Google Sheets?

3. Does it encourage repeat visits?

4. Does it contribute to monetization?

5. Can it be reused?

6. Is it API-driven?

7. Can AI eventually operate it?

If any answer is No,

improve the design first.

---

# Administrator Philosophy

The administrator should never wonder:

"Which file should I edit?"

Everything should be manageable through Google Sheets.
